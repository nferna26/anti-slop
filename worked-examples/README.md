# worked-examples/

End-to-end runs of the methodology against real (or synthetic but realistic) decisions. Each worked example walks from raw source material through cards, canon, and execution, showing how citations propagate and how the evidence / judgment / unknowns split shows up in practice. They exist because the schema rules are abstract and an operator's first question is "what does a real run look like?" Currently empty; examples accrete here as they pass review. See `methodology/schema.md` for the three-way body split a worked example must preserve.
