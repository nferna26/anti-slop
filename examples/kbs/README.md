# examples/kbs/

Reference KBs that follow this methodology end-to-end. Each one is a complete (small) knowledge base demonstrating the five-layer structure, the nineteen-field frontmatter, and the cross-reference vocabulary on a real domain. They exist so an operator can see what the methodology looks like at rest before they build their own — a working KB is shorter than an explanation of one. The schema these KBs conform to is in `methodology/schema.md`. Currently empty; example KBs land here when they are cleared for public review.
