# skills/

Each Skill is a folder with a `SKILL.md` file (YAML frontmatter plus body instructions) and any scripts the instructions call. The directory holds Skills as a pack — the installable surface between this methodology and an operator's own knowledge base. A Skill encodes one repeatable operation (ingest, query, lint) so the rules in `methodology/schema.md` can be exercised by an agent or by hand without restating them. Currently empty; Skills land here as they ship.
